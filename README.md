🩺 Diabetes Risk Predictor

A machine learning web application that predicts diabetes risk using multiple models and provides an interactive, user-friendly interface built with Streamlit.

🚀 Features
🔍 Predict diabetes risk based on patient inputs
🌲 Uses Random Forest, Linear Regression, and Decision Tree models
📊 Displays average risk score and individual model predictions
📄 Separate detailed analysis page for model-wise breakdown
🎨 Clean and modern UI built with Streamlit
⚡ Fast and interactive experience
🧠 Machine Learning Models Used
Random Forest Regressor
Linear Regression
Decision Tree Regressor

The final prediction is calculated as the average of all three models.
